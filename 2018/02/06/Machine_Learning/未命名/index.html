线性回归

发表于 2018-02-06 | 分类于 Machine Learning |

房价的问题

假设我们现在已知一些房屋面积的大小和价格之间的关系(见下图),我们将如和去预测某个面积的价格，一个简单的方法是我们画一条直线，这条直线尽可能多的经过数据所分布的区域，然后根据这条直线来预测房屋价格。在机架学习中种问题具有以下特征：

  • 样本中每一个输入值都有一个正确的输出值：监督学习
  • 预测一个具体的输出值：回归

以上就解开了我们这片文章的内容：监督学习-回归。接下来我们将用数学的方法教你如何做到”尽可能“

符号的意义

为了方便我们进行学习,我们首先要对一些公式符号进行约约定，参考实例数据集：

房间大小（input） 价格（output）
2014 460
1416 232
1534 315
….. ….
  • x(i)： 表示第i个输入值 。例如x(2)=1416
  • y(i)：表示第i个输出值。例如y(3)=315
  • (x(i),y(i))： 表示一个训练样本。例如(x(2),y(2))=（1416,232）
  • m：表示训练样本的个数。表格中训练集合的个数就是m

有了上面的约定后，我能再来规范一下如何去描述我们所要解决的问题：

given a training set, to learn a function h : X → Y so that h(x) is a “good” predictor for the corresponding value of y（给定一个数据集合，让机器学习算法去得到一个方程h，这个方程可以根据输入的x来预测输出的y）

对于初学者我们将暂定这个方程式为一下方程，对的你没看错，就是一个非常简单的线性函数，当然这个方程也可以是一个更加复杂的方程，但是我们从这里起步学习，我们称下方程模式为线性回归模型（linear regression)，他还有另外一个名称叫单变量线性回归，这个单变量就指
$$
hθ(x)=θ0+θ1*x
$$

代价方程

什么是代价方程

到底要怎么去尽可能的拟合和这直线和数据点？这个尽可能要做的事情就是得到一条直线，这条直线到所有已知点的距离平方和最小为，如果用J(θ0,θ1)来代表这个值，则J(θ0,θ1)可以表示为
$$
J(\theta_0, \theta1) = \dfrac {1}{2m} \displaystyle \sum {i=1}^m \left ( \hat{y}{i}- y{i} \right)^2 = \dfrac {1}{2m} \displaystyle \sum {i=1}^m \left (h\theta (x{i}) - y{i} \right)^2
$$
这个方程就是代价方程，很明显当直线穿所有点的时候J(θ0,θ1)=0，我们的目的就是通过调整θ0,θ1，求这个方程的最小值。
